# VTuber Comment Viewer

Solid.js + Vite + Vinxi + Solid Start を使用したコメントビューアー

## セットアップ

```bash
npm install
```

## 開発

```bash
npm run dev
```

## ビルド

```bash
npm run build
```

## Firebase Hosting にデプロイ

プロジェクトのルートディレクトリで:

```bash
firebase deploy --only hosting
```

または Firestore のルールも一緒にデプロイ:

```bash
firebase deploy
```
